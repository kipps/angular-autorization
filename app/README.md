## Development server
Move to server folder and run `npm install`. After that you can run `node ./server.js` into server folder or return back to root folder and run `npm run server`.

Also you need run `npm install` for root app folder. After that you can run `ng serve` or `npm run start` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
